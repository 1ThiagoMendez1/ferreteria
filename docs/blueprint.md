# **App Name**: Hardware Hub

## Core Features:

- Product Catalog: Browse and search for products by category and keywords.
- Inventory Management: Add, edit, and delete product listings.
- Stock Quantity Adjustment: Manually adjust product quantities or automatically update quantities based on sales (manual update only for MVP).
- Stock Alerts: Define a 'minimum stock' level for each product and receive alerts when the stock falls below this threshold.
- Categorized Inventory: Group products by category for easy browsing and management.
- Location-Based Inventory: Assign products to specific store locations (not tracked for MVP, but will be user-configurable).
- AI Product Description Tool: Use generative AI tool to create or refine product descriptions, helping ensure listings are informative and engaging.

## Style Guidelines:

- Primary color: Deep blue (#34495E) to convey reliability and professionalism.
- Background color: Light gray (#ECF0F1) for a clean and modern feel.
- Accent color: Orange (#E67E22) for calls to action and important alerts.
- Body and headline font: 'PT Sans' for a modern and readable style.
- Use simple and clear icons to represent product categories and actions.
- A clean, grid-based layout to display products and information clearly.
- Subtle transitions and animations to enhance user experience during navigation and interactions.
